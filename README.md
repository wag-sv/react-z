# Re.mo App

## ABOUT THE APP

_REMO_ is an application that shows host availability status information on a map [OpenStreetMap](https://www.openstreetmap.org/#map=14/-20.4595/-54.5942). The information is brought from the [Zabbix](https://www.zabbix.com/community) API.
